# MyTodoApp

**Yeast 세미 프로젝트**는 베이커리 회사 내의 입출고, 매출, 인력 관리 등 다양한 실무 업무를 효율화하기 위해 제작한 **ERP 시스템**입니다.  
Spring Boot 기반으로 설계된 이 프로젝트는 팀원 간의 **협업, 요구분석, 시스템 설계, UI/UX 디자인, 기능 구현, 테스트**까지  
실제 프로젝트의 전 과정을 경험하며 완성하였습니다.
